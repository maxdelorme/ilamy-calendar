## Notes

Edit callbacks (generated instance, scope `this`): `onEventUpdate` → base (`id` = base, `uid` = series, updated `exdates[]`); `onEventAdd` → override (`{baseId}_modified_*`, same series `uid` unless `updates.uid` overrides). Scope `this` on stored override → `onEventUpdate` only. Scope `following` → `onEventUpdate` + `onEventAdd` (`_following`). Scope `all` → `onEventUpdate` once; remove override rows in your store (same `uid`, no `rrule`).

`renderHook`: keep `events` (and other config objects) referentially stable unless testing prop sync — inline `events: [base]` retriggers the `events` effect and can cause `Maximum update depth exceeded`.
